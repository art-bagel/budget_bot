
# Frontend Feature Checklist

Функциональность, которую поддерживает текущая БД и бэкенд, но которая ещё не реализована на фронтенде.

---

## 1. Семья (Family)

Бэкенд: `/api/v1/family/*` — полностью готов, фронтенд — реализован.

- [x] Страница/раздел "Семья" — отдельная вкладка в навигации (`pages/Family.tsx`)
- [x] Создать семью (POST `/family`) — ввод названия
- [x] Отобразить текущую семью пользователя (GET `/family/me`)
- [x] Список участников семьи (GET `/family/members`) — имя, роль
- [x] Пригласить пользователя по @username (POST `/family/invite`) — только для владельца
- [x] Список входящих приглашений (GET `/family/invitations`)
- [x] Принять приглашение (POST `/family/invitations/{id}/accept`)
- [x] Отклонить приглашение (POST `/family/invitations/{id}/decline`)
- [x] Индикатор/бейдж при наличии ожидающих приглашений (красная точка на вкладке)
- [x] Покинуть семью — кнопка для участника (POST `/family/leave`, новый backend эндпоинт)
- [x] Распустить семью — кнопка для владельца (POST `/family/dissolve`, новый backend эндпоинт)
- [ ] Исключить участника из семьи — требует новой DB-функции (нет в storage)
- [ ] Отмена отправленного приглашения — требует новой DB-функции (нет в storage)
- [ ] Список отправленных приглашений с их статусами — требует нового DB-запроса (нет в storage)

---

## 2. Банковские счета (Bank Accounts)

БД: таблица `bank_accounts` поддерживает несколько счётов на пользователя/семью; бэкенд возвращает список, но создание/редактирование счётов не реализовано на фронте.

- [ ] Страница управления счетами (список всех счётов пользователя)
- [ ] Создать новый банковский счёт (название, признак основного)
- [ ] Переименовать банковский счёт
- [ ] Сменить основной счёт (is_primary)
- [ ] Архивировать/деактивировать счёт
- [ ] Просмотр архивированных счётов (GET `/bank-accounts?is_active=false`)
- [ ] Выбор конкретного счёта в диалоге расхода/дохода (сейчас жёстко привязан к одному)
- [ ] Семейные банковские счета — отображение и переключение между личными и семейными
- [ ] Мультивалютный баланс счёта — детальный просмотр (breakdown по валютам + historical cost)

- [ ] Редактирование состава группы прямо с дашборда (замена участников, изменение долей)
