# Dashboard — концепт v2 (Tinkoff-leaning)

Второй вариант Dashboard в рамках редизайна Mini App. Ближе к визуальной системе Тинькофф, но без буквального копирования.

## Как открыть

Достаточно двойного клика по `index.html` — сборка не нужна, всё работает на ванильном HTML/CSS/JS. Рекомендуется открывать в окне шириной ≈375–414px (DevTools → Toggle device toolbar → iPhone 14 Pro / Pixel 7), чтобы увидеть мобильный макет как в Telegram Mini App.

## Что отличает v2 от v1

| | v1 (warm paper) | v2 (Tinkoff-leaning) |
|---|---|---|
| Фон | Тёплый кремовый #F1EBDC | Прохладный серый #EEF0F3 / плотный чёрный #0A0A0B |
| Акцент | Ember-оранжевый на ink-hero | Фирменный жёлтый #FFDD2D на чёрном тексте |
| Шрифт | Golos Text + Fraunces italic | Onest — геометричный, плотный |
| Hero | Ink-карточка, редакционная подача | Жёлтая карточка с breakdown счетов |
| Счета | Вертикальный список в hero | Breakdown-строки в hero (Личный / Семейный / Инвестиции / Кредиты) |
| Quick actions | В углу hero | Отдельной строкой под hero (как у Т-банка) |
| Категории | Иконки через border-чип | Цветная плитка-квадрат (g/o/b/p/r/v тинты) |
| Табы | Минималистичные метки | Активная со жёлтым underline |

## Интерактив

- **Тема** — переключатель солнце/луна в хедере. Запоминается в `localStorage` (`budget-theme-v2`).
- **Hero — «− кредиты / + кредиты»** — чип в углу карточки меняет total (2 647 300 ↔ 2 827 300) и плавно затемняет цифру на время перехода.
- **Категория** — тап по плитке открывает bottom-sheet с деталями: имя, иконка, тинт, остаток и принадлежность к группе подставляются из `CAT_DATA` в `script.js`.
- **Пополнить** — кнопка в quick-actions открывает sheet пополнения (моки выбора счёта, суммы и типа).
- **Свободный остаток** — тап по карточке открывает sheet перевода между «Личный ↔ Семейный».
- **Табы** — переключают активный индикатор (без реальной навигации).
- **Клавиатура** — `Esc` закрывает активный sheet; категории фокусируемы (Enter/Space).

## Структура

```

├── index.html  — разметка, встроенный SVG-sprite с иконками
├── styles.css  — токены (light/dark), компоненты, анимации входа
```

## Что намеренно опущено

- Реальные API/данные (моки захардкожены)
- Навигация между страницами (табы — визуальные)
- Формы внутри sheets — демонстрационные, не валидируют ввод
- Жесты драга для закрытия sheet (закрывается по backdrop/крестик/Esc)

Перенос в `frontend/` начинаем только после выбора направления.
