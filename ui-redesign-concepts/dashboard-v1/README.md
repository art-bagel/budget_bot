# Dashboard — концепт v1

Интерактивный прототип редизайна главного экрана в формате Telegram Mini App.

## Как открыть

Дважды кликнуть по `index.html` — откроется в браузере. Сборка не требуется.

Для mobile-вида: включить DevTools (Cmd+Opt+I) → toggle device toolbar → iPhone 14/15 (390 × 844).

## Что работает

- **Тема** — кнопка в правом верхнем углу. Сохраняется в localStorage.
- **Toggle «± кредиты»** — пересчитывает чистый капитал в hero-карточке.
- **Категория** — тап по любой из карточек открывает bottom-sheet с деталями и последними операциями.
- **Пополнить** — быстрая кнопка в hero открывает sheet дохода.
- **Свободный остаток** — тап открывает sheet перевода.
- **Табы снизу** — подсвечиваются по клику (навигация не настоящая).

## Дизайн-направление

- Палитра: тёплая бумага (`#F1EBDC`) + ink-hero (`#17140F`) + ember-акцент (`#CB5F23`)
- Шрифты: Golos Text (UI, Cyrillic-native, humanist) + Fraunces italic (editorial accent на hero eyebrow)
- Табличные цифры везде, где цифры
- Скруглённые углы варьируются: 6 → 28 px, не «всё подряд 24px»
- Кастомные line-иконки (inline SVG), без эмодзи
- Subtle paper grain через SVG noise в фоне
- Dark theme — настоящий warm black (`#141210`), не navy
- Тонкая амбер-рамка на hero создаёт «журнальный» штамп

## Файлы

- `index.html` — разметка + SVG-спрайт иконок
- `styles.css` — дизайн-токены (light/dark), компоненты
- `script.js` — интерактивность
