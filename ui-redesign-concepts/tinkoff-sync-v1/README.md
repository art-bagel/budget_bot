# Tinkoff Sync — concept v1

Замена двух модалок на right-side slide-over в едином стиле сервиса:

- `TinkoffConnectDialog` — подключение токена + маппинг счетов
- `TinkoffSyncDialog` — предпросмотр операций (вводы / выводы / автооперации) с разрешениями

## Демо

Открыть `index.html` локально. На стартовом экране — карточка инвестиционного счёта с кнопками:

1. **Подключить** → запускает Connect-flow (3 шага: токен → выбор счетов → итог)
2. **Подтянуть данные** → запускает Sync-flow с табами «Вводы / Выводы / Автоматические» и bulk-actions

## Дизайн-решения

- **Slide вместо модалки.** Right-to-left full-height панель: больше места под длинные списки операций, аккуратные sticky header/footer, понятная навигация назад.
- **Warm paper + ember accent** — наследует токены из `dashboard-v1`.
- **Golos Text** + **Fraunces italic** для eyebrow-меток.
- **Радио-карточки** для разрешений (вместо узкого `<select>` сразу видны все варианты).
- **Bulk actions** — пилл-кнопки сверху списка, для массового назначения.
- **Inline-чипы** баланса и предупреждений — без всплывающих popover.
- **Тёмная тема** — переключатель в шапке концепта.

## Что портировать в `frontend/` после ревью

- `slide-over` каркас (overlay + panel + анимация) → переиспользуемый компонент `<SlideOver>`.
- Радио-карточки разрешений `.res-card`.
- Bulk-actions ряд `.bulk`.
- Степпер `.steps` для многошаговых флоу.
