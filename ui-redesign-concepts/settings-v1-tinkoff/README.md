# Settings — концепт v1 (Tinkoff-leaning)

Редизайн страницы «Настройки» в едином языке с `dashboard-v2-tinkoff` / `portfolio-v1-tinkoff` / `credits-v1-tinkoff`: прохладный серый фон, жёлтый акцент `#FFDD2D`, шрифт Onest, плотная подача. Статичный прототип, без сборки.

## Как открыть

Двойной клик по `index.html`. В DevTools → Toggle device toolbar → iPhone 14 Pro / Pixel 7 (375–414 px).

## Главные дизайн-решения

### Identity card вместо большого hero-числа

У настроек нет «героя-цифры», поэтому вместо привычной жёлтой/ink-карточки с балансом — **компактный ink-блок личности**: монограмма-аватар с жёлтым свечением, имя, @handle, и три chips (`Владелец семьи / Базовая ₽ / 4 в семье`). Цвет тот же ink, что и hero других страниц — визуальная связка сохраняется.

### Tabs со скроллом и слайдящим жёлтым индикатором

6 вкладок не помещаются в 440px без сжатия — поэтому таб-бар горизонтально скроллится с snap-points. Под активной вкладкой — тонкий жёлтый underline (`tabs__ind`), который анимированно перетекает между табами. Справа — мягкий fade-градиент, намекающий на скролл.

### Theme picker — визуальные превью

Не три сухих кнопки, а **3 мини-карточки 78px** с реальными превью темы:
- **Системная** — диагонально разрезанная пополам, светлая + тёмная половины
- **Светлая** — серый фон + белая мини-карточка + жёлтый chip-акцент
- **Тёмная** — чёрный фон + ink-карточка + жёлтый chip

Активная подсвечивается жёлтой рамкой + жёлтым фоном (yellow-tint), в правом верхнем углу — жёлтый чекбокс с галкой (анимированный scale-in).

### Опасная зона — отдельный блок, не строка

`Удалить аккаунт`, `Распустить семью` вынесены в отдельную карточку с **красной полосой слева** (`border-left: 3px solid neg`), подзаголовком в caps + красная точка-буллет, иконкой-предупреждением в красном tint. Кнопка `btn--danger` — мягкая (red-tint фон + красный текст), на hover становится залитой красной. Это считывается строже, чем простой красный текст в строке.

## Структура страницы

1. **Bar** — назад + «Мини-приложение / Настройки» + theme toggle.
2. **Identity card** — ink-карточка с аватаром и chips.
3. **Tabs** — 6 вкладок (`Оформление · Аккаунт · Семья · Инвестиции · Интеграции · Данные`).
4. **Pane** — соответствующий выбранной вкладке (см. ниже).

### Tab content

#### Оформление
- **Тема интерфейса** — theme grid (system / light / dark) с превью.
- **Поведение** — три свича: `Подсказки жестов`, `Компактный список`, `Тактильный отклик`.

#### Аккаунт
- **Контуры и валюты** — личная база, семья, семейная база, банковский счёт (последний — навигационный, с шевроном).
- **Опасная зона** — `Удалить аккаунт`.

#### Семья
- **«Костенко»** — owner-pill `Пригласить`, список из 4 участников с monogram-аватарами в разных тонах + role-badge (`Владелец / Участник / Pending`). Pending-участник — пунктирный аватар, опуская opacity.
- Ссылка `История семьи →`.
- **Опасная зона** — `Распустить семью`.

#### Инвестиции
- Список из 3 mock-счётов как `invest-tile`: brand-chip с тинтом + название + meta (контур · тип · провайдер) + ID-tag + delete.
- Inline-форма создания с полями `Название`, `Контур / Тип актива`, `Провайдер` — раскрывается по `+ Новый`.

#### Интеграции
- **Подключения** — три provider-card:
  - **Тинькофф Инвестиции** (активный) — жёлтый brand-blob, статус «подключено», список одного счёта (Брокерский · 2110), action `+ Добавить ещё счёт`.
  - **Сбер · СБП** (`prov--soon`) — приглушённый, badge `Скоро`.
  - **Госуслуги** (`prov--soon`) — то же.
- **API-токены** — заглушка с подсказкой про шифрование + кнопка `+ Токен`.

#### Данные
- **Экспорт** — `Операции · CSV` (~2,4 МБ), `Категории и группы · JSON`, `История курсов` (CBR · CC).
- **Кэш и пересчёт** — размер локального кэша (312 КБ), кнопка `Пересчитать остатки`.
- Внизу — версия и сборка.

## Интерактив

- **Тема (header)** — переключатель в шапке. Сохраняет в `localStorage: settings-theme-pref-v1`.
- **Theme grid** — клик по карточке меняет тему. Учитывает `prefers-color-scheme` для `system`.
- **Tabs** — клик переключает pane (fade-in). Активная скроллится в центр. Сохраняет в `localStorage: settings-active-tab-v1`.
- **Switches** — анимированный thumb с overshoot, vibrate(8) если включён `Тактильный отклик`. Сохраняет в `localStorage: settings-toggles-v1`.
- **Invest form** — `+ Новый` toggle открытия. Esc закрывает. Submit показывает toast.
- **Опасные действия** — `confirm()` + toast.
- **Toast** — короткое подтверждение всех действий, появляется снизу по центру (1.8 сек).

## Открытые вопросы

1. **Identity card vs. большой hero** — нужен ли тут вообще hero, или достаточно бара? Сейчас identity-карточка даёт визуальную связь со всей системой и содержит всё ключевое (роль, базовая валюта, размер семьи) одним блоком.
2. **Theme picker — три превью или один сегмент** — мини-превью красивее, но занимают больше места. Альтернатива: компактный сегментный переключатель (Sys/Light/Dark) на одной строке.
3. **Что в «Поведение»** — добавил `Компактный список` и `Тактильный отклик` как осмысленные опции. Стоит ли держать сейчас, или это излишнее обещание UI, которого ещё нет?
4. **Скоро в интеграциях** — `Сбер СБП` и `Госуслуги` — реальные ли это направления, или только пример? Если нет планов — убрать, оставить только Тинькофф.
5. **API-токены** — нужны ли как отдельный блок, или это часть конкретного провайдера? Сейчас как «заготовка», без UI создания.
6. **Экспорт** — JSON-бэкап структуры — в реальности есть такой эндпоинт, или только CSV операций?
7. **Опасная зона** — стоит ли тут красная-полоса-слева, или хватит просто красного бордера снизу. Полоса слева — заметнее.
